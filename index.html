>_ mySideProjects_

TrendSnap - Podcast Viral Clip Extractor ↗

An intelligent script that identifies and extracts the most engaging 1-minute clips from podcast videos using audio energy, sentiment analysis, and scene detection. Built with MoviePy, OpenCV, Librosa, NLTK, and more, it computes a virality score for each clip and exports the top ones for use on platforms like Instagram Reels, TikTok, and YouTube Shorts.

IndicF5 Voice Cloning ↗

A multilingual voice cloning project using AI4Bharat’s IndicF5 model. It generates speech in Hindi, Marathi, Tamil, and Telugu, mimicking the tone of reference audio. Ideal for voiceovers and regional language applications.

Spotify Playlist Downloader ↗

A Python-based tool to download entire Spotify playlists as MP3s using spotDL and FFmpeg, with live terminal progress and organized output.
